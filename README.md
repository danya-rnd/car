## Установка зависимостей

Запускаем команду для установки зависимостей в папку node_modules:
```
npm install
```
## Запуск

Запускаем на отслеживание изменений и пересборку с Browsersync на http://localhost:3000:
```
npm run start
```

## Пересборка

Сборка проекта в папку build
```
npm run build
```
